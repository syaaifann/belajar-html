**HTML** adalah singkatan dari *HyperText Markup Language*. Ingat, HTML **bukan** bahasa pemrograman, melainkan bahasa *markup* standar untuk membuat struktur halaman web. Jika website adalah sebuah bangunan, maka HTML adalah kerangkanya.

## âœï¸ Elemen Teks

Bagian ini mencakup cara mengelola teks dalam HTML:

* **Heading:** `<h1>` sampai `<h6>` untuk judul dan sub-judul.
* **Paragraph:** `<p>` untuk blok teks biasa.
* **Formatting:**
* <u>Underline</u> menggunakan tag `<u>`.
* Subscript: H<sub>2</sub>O menggunakan tag `<sub>`.
* Superscript: E = mc<sup>2</sup> menggunakan tag `<sup>`.
* **Extra Text:** `<strong>` (tebal), `<em>` (miring), `<mark>` (highlight), dan `<small>` (teks kecil).
